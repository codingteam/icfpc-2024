icfpc-y2024
===========
Contest page: https://icfpcontest2024.github.io/

Prerequisites
-------------
Install [Stack][stack] v2.15.7.

Compilation
-----------
```console
$ stack build
```

Running
-------
```console
$ stack exec icfpc-y2024-exe
```

Testing
-------
```console
$ stack test
```

[stack]: https://docs.haskellstack.org/en/stable/
