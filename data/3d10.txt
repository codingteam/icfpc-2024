# Input
  A number `A` consisting of digits `1` (representing `(`), `2` (representing `)`), `3` (representing `[`) and `4` (representing `]`), at most `40` digits long.

# Output
  `1` if the parentheses from the input are properly balanced, and `0` otherwise.

# Example
  * `A = 134212`
    `Answer = 1`
  * `A = 1132`
    `Answer = 0`
